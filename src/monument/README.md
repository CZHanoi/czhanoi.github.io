## 4月22日

连日来，心头似笼着一层化不开的薄雾，阴沉了好几个日夜。白昼里步履匆匆，夜深时辗转难眠——已是第三个这样的夜晚了，枕边的月光换了又换，思绪却始终绕着同一个地方打转。为了躲开心里那点悬而未决的惦念，我一头扎进了工作里，用一份又一份文件、一条又一条待办，给自己筑起一道暂时的堤坝，好让那些翻涌的情绪不至于漫出来。

而今天，我终于悄悄跨过了那道在心里盘桓许久的门槛。那是一件极小的事，小到不值一提，却又重得让我踌躇了整整数日——如今，某个名字静静地躺在了我的联系人列表里，带来了久违的梦的甜美，一种无法用语言形容的感觉遍布全身，聚集的能量在我胸腔仿佛顷刻就要爆发，这真是不可思议……

窗外的天依旧阴着，云层低垂，迟迟不肯散去。可奇怪的是，心里那片积压多日的郁色，竟随着这一步的迈出，悄然淡了许多。原来困住我的从来不是外面的阴雨，而是自己迟迟不敢落下的那一步。

前路仍有许多未知，像一页尚未展开的信笺，不知会写下怎样的故事。但无论如何，我愿意去试一试，去争取，去认真地对待这份微小而珍贵的心动。

加油吧。希望Muse眷顾我。



### 4月23日

今天仍是步履不停的一日，案头的事务一件接着一件，时间像指缝间的流沙，不经意便已是黄昏。

最让我心头微暖的，是那份小小的心意终究被收下了。那一刻，仿佛有一枚轻羽悄然落在水面，荡开一圈极浅的涟漪——虽细微，却真实地泛过了心底。只是涟漪过后，湖面复又归于平静，平静得让我隐约读懂了些什么。有些目光虽温和，却终究隔着一层看不见的薄纱；有些回应虽礼貌，却并不通向我所期盼的那个方向。说不清是从哪个细节察觉的，大约是那种直觉吧——像春夜里忽然飘来的一缕风，不冷，却让人清醒。

但这些，已经不那么重要了。

重要的是，我迈出去了，我认真过了，我没有辜负自己那份郑重其事的心意。至于后来的事，交给时间，也交给缘分便好。

天气预报说，明天会是个晴天。

那就趁着这份晴朗，把自己重新梳理一遍吧。体重要一点一点减下去，科研的课题要一步一步推进，那些搁置许久的书、听了一半的音乐、想去却未曾启程的地方，也该慢慢拾起来了。生活本就不该只围着一个人打转，一个丰盈的自己，才是这一切的底色。

我仍不打算放弃——不是执拗地守着某个结果，而是执着地相信，每一分努力都不会白费。愿未来的我，比今天更从容，更明亮，也更值得被温柔以待。

将`hgWGCNA`挂上，晚安。



### 4月24日-4月27日 淳安-千岛 感言录

- 绝对没办法原谅我自己

- 如果得不到其他人的认可，何以为人；尽可能的让更多人认可吧

- 还是满脑子都在想着Muse，唉……



​	出发前的那一夜，我细细收拾行装，挑了些自以为"拿得出手"的吃食，一样样仔细码进包里，仿佛把一份小小的郑重也一并装了进去。可旅程结束时才发现，那些精心挑选的食物，竟又原封不动地随我返程——大约这一路，心思早已不在唇齿之间了。

​	启程那日，与友人（Chione）同行。Chione在路上随口讲起自己的近况，进而也聊起了她——Muse女神，我听着听着，竟有种天光骤然黯淡的错觉。一方面，悟已往之不谏——过往那些笨拙的、莽撞的、不够妥帖的瞬间，早已悄悄沉淀在某些人的眼底，化作一层难以洇开的墨痕，尤其在Muse的眼中，我不知道已经留下了怎样负面、丑陋的印象；另一方面，我心里清楚，要稀释那一层墨，怕是要走好长好长的路。仍是决意继续走下去的，只是对于结局，已不敢再抱什么笃定的期许了……

​	搞笑的是，在高铁上恰巧坐在我旁边的是实验室的师兄，我和师兄聊了一路也没有想起师兄的名字xs；从常规的课题询问，到后面研究生的住宿事宜，再到后面的薪资工作问题，话匣一旦打开就收不回去了，整整唠了一路，好不容易积攒的社交能量还没下高铁就已经消耗了一大半。

​	后面就是商量在哪里happy了。从4个小资历的密谋，非常幸运地得到了Chione的关注；Chione有着独特的号召力，这让我们的自闭小登团队迅速壮大为带走组里1/4人群的大队伍；旅行的过程是美好而又欢快的（好在社交能量还剩下一些），但同行人随手为我留下了几张照片，归来翻看时，镜头里的那个身影让我怔了一怔——那一瞬，仿佛有什么东西在心里轻轻沉了下去，连指尖也跟着凉了几分。照片中的我<span style="color: red; font-weight: bold;">bloated, ugly, disgusting</span>，尤其是在一众同行人中显得额外扎眼，<span style="color: red; font-weight: bold;">我真的讨厌我自己</span>……合上屏幕，完全不再敢看第二眼。唉……对自己的讨厌又一次达到了顶点，尽管我心里知道实验室的同门不会因为我丑陋而讨厌我，但因此产生的社交意愿度下降很难量化但绝对真实存在。

​	到了夜晚的餐桌前，轮到自我介绍时，还是放弃了选择过于哗众取宠的发言。一方面真的不想让大家记住现在丑陋的我，另一方面，紧绷的情绪让我又有些语无伦次：话音落下才察觉语序乱了，重点散了，余下的只是有些发烫的耳廓和未尽之意。唉，就这样吧，如何能够从容地面对众人的目光（尤其是领导面前），这门功课，看来还得再细细练习——身材外貌、语言交流、社交正念，唉这个话题就写到这吧……

​	第二天的内容，后面简简单单的放些风景照吧。

​	总结就是，即便Muse不会眷顾我，而Muse带来的正向促进机会绝对不能从指尖流走；即便我没有投入Muse怀抱的幸运，我至少也要达到那个距离，达到应有的高度，获得应有的能力；**这不是在喊口号，而是救己图存的革命。**
